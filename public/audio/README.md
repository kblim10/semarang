# Lagu tema

Taruh file audio lagu tema di folder ini dengan nama:

```
theme-song.mp3
```

Ini menggantikan embed Spotify lama (yang berat dan nggak bisa
di-restyle) dengan tag `<audio>` bawaan browser - jauh lebih ringan
dan sepenuhnya bisa kita kontrol tampilannya.

Setelah file-nya ada, edit judul & nama artis di
`src/data/content.ts` (variabel `themeSong`).

Catatan penting: pastikan kamu punya hak pakai atas file audio yang
diunggah (misalnya rekaman sendiri, lisensi royalty-free, atau lagu
yang memang kamu punya izin untuk dipakai). Mengunduh audio dari
Spotify/YouTube untuk dipakai ulang tanpa izin melanggar hak cipta,
jadi itu bukan sesuatu yang bisa saya bantu lakukan secara otomatis.
